# Non-Coding-Variation-Browser 
## Description
This project is to aide uses in viewing single nucleotide polymorphisms(SNPS) over CHIP-Seq/DNAse data from Encode(https://www.encodeproject.org/). 
## Dependencies
- python 2.7.X
- django module
- tqdm module
- pandas module
- sqlite3 module
- gzip module
